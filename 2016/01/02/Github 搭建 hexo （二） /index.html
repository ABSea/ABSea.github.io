Github 搭建 hexo （二）

2016-01-02 | 分类于 github blog

在github page 上显示helloworld，这次不是本地效果咯。

前提条件：

  • github账号
  • 本地git环境

step 1：

  • 创建一个仓库，名字以自己的github名字开头,即yourName.github.io

step 2：

  • 把上面的仓库克隆到自己的电脑一个自定义的目录，

step 3：

  • 进入克隆后的目录。
  • 创建一个html文件，并写入内容。比如使用 echo "hello world" > index.html

step 4：

然后commit到 github 上，使用 git commit -m "commit message"

step 5：

使用git push -u origin master，提交到github服务器

这样之后就可以进入http://yourname.github.io/，将yourname换成你的github名字就行了，查看helloworld的效果了，这里就不展示效果了。
